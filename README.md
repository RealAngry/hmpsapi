# 🔐 Login API – Authentication System for Web Projects

This is a secure and scalable **Login API** built using **Node.js**, **Express**, and **TypeScript**, designed to handle authentication for modern web applications. It includes full support for user registration, login, JWT-based sessions, and input validation.

---

## 🚀 Features

- ✅ User Registration
- 🔐 Login with JWT Authentication
- 🔁 Token Refresh endpoint (optional)
- 🔒 Secure password hashing using bcrypt
- 📜 Input validation with `zod` / `express-validator`
- 🧑‍💻 Built with Express + TypeScript
- 🌐 CORS support for frontend integration
- 🗃️ MongoDB for storing user credentials

---

## 🛠 Tech Stack

- **Node.js** + **Express**
- **TypeScript**
- **MongoDB** (via Mongoose)
- **bcryptjs** – Password hashing
- **jsonwebtoken** – Auth tokens
- **zod** or **express-validator** – Input validation
- **dotenv** – Environment variables
